# Changelog

All notable changes to DocQuery are documented here.

## [0.1.0] - 2026-07-11

### Added

- Workspace-scoped document upload (PDF, DOCX, TXT) with background indexing
- RAG chat with sentence-level citations and confidence badges
- Auth: email/password (Argon2id) + Google/GitHub OAuth
- Trace Rail UI, command palette, pinned/recently viewed documents
- Conversation Markdown export and analytics dashboard
- Marketing pages with SEO (sitemap, robots, OG, JSON-LD)
- CI workflow: lint, typecheck, test, build
